# Visual QA Checklist

- [x] PNG opened and inspected.
- [x] No text overlap.
- [x] No clipped or truncated labels.
- [x] GIF has visible motion.
- [x] Excalidraw opens as editable JSON.
- [x] `--check` returned `ok: true`.
- [x] Final response includes GIF, PNG, Excalidraw paths, dimensions, frame count, FPS, and warnings.
